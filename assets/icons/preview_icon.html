ABRA - Auto Body Repair Assistant

Новая иконка приложения в различных размерах

48x48 (Android mdpi)
72x72 (Android hdpi)
96x96 (Android xhdpi)
144x144 (Android xxhdpi)
192x192 (Android xxxhdpi)
512x512 (Web)

Инструкция по установке

  1. Конвертируйте SVG в PNG используя любой онлайн конвертер или графический редактор
  2. Сохраните PNG как app_icon_new.png в папке assets/icons/
  3. Обновите pubspec.yaml: 
    flutter_launcher_icons:
  android: true
  ios: true
  image_path: "assets/icons/app_icon_new.png"
  adaptive_icon_background: "#4A90E2"
  adaptive_icon_foreground: "assets/icons/app_icon_new.png"
  4. Запустите команду: flutter pub run flutter_launcher_icons
