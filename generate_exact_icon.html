Инструкция:

  1. Нажмите на размер 1024x1024 или кнопку "Скачать все размеры"
  2. Сохраните файл как app_icon.png в папке assets/icons/
  3. Запустите: flutter pub run flutter_launcher_icons
  4. Очистите кэш: flutter clean
  5. Пересоберите приложение: flutter run
