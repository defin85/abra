Инструкция:

  1. Нажмите на нужный размер иконки выше для скачивания
  2. Сохраните 512x512 как app_icon_new.png в assets/icons/
  3. Запустите в терминале: 
    flutter pub get
flutter pub run flutter_launcher_icons
  4. Для ручного обновления скопируйте файлы в соответствующие папки:
    • 48x48 → android/app/src/main/res/mipmap-mdpi/launcher_icon.png
    • 72x72 → android/app/src/main/res/mipmap-hdpi/launcher_icon.png
    • 96x96 → android/app/src/main/res/mipmap-xhdpi/launcher_icon.png
    • 144x144 → android/app/src/main/res/mipmap-xxhdpi/launcher_icon.png
    • 192x192 → android/app/src/main/res/mipmap-xxxhdpi/launcher_icon.png
    • 192x192 → web/icons/Icon-192.png
    • 512x512 → web/icons/Icon-512.png
